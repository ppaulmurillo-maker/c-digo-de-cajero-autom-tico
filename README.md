# c-digo-de-cajero-autom-tico
En este pequeño proyecto presento la codificación de el software de Cajero automático del Banco del Pacifico.
En en este codigo que he generado exiten dos usuarios de dicho bancos de los cuales poseen tarjetas del banco del Pacifico y montos diferentes de dinero en sus respectivas cuentas al cual podran ingresar identificandose con su usuario y pin de contraseña de 4 digitos, podran hacer 
1. Consulta de saldos
2. Retiro de dinero
3. Deposito de dinero.
